# instaskeleton example

This folder contains the local React + Vite showcase for `instaskeleton` using HeroUI.

## Run

From repository root:

```bash
npm run example:install
npm run example:dev
```

## What this demo is for

This is the visual reference app for the package. It should answer two questions:

1. What kinds of UI can inference mode cover reasonably well?
2. When manual schema is used, how closely does the skeleton mirror the inner component layout?

## Covered scenarios

1. Profile rows and dashboard cards
2. Product grids and pricing layouts
3. Social feed cards and chat threads
4. File lists and data tables
5. Mixed settings forms with inputs, selects, switches, sliders, and progress
6. HOC wrappers with preconfigured skeleton behavior
7. Primitive node references and animation comparison
8. Side-by-side component anatomy examples for manual schema
